# Предисловие

Практически весь nvim config украден с https://github.com/gunlinux/dotfiles

Там написано, что часть украдена с https://github.com/mitsuhiko/dotfiles

(мб конфиг nvim не стырен, но все же)

# Установка

Запустите set_config.sh:
```bash
bash set_config.sh
```

**Важно:** этот скрипт удалит и заменит ваши конфиги, поэтому лучше сохраните их заранее
